Areas of interest include:

## Current Projects

### Lumen

An open-source AI orchestration platform focused on session continuity, context management, provenance, and model-agnostic workflows.

Rather than replacing existing AI models, Lumen provides the engineering infrastructure around them—helping preserve knowledge across sessions, maintain project continuity, and make AI-assisted development more transparent and reproducible.

Project philosophy:

* Model agnostic
* Open architecture
* Knowledge preservation
* Engineering-first design
* Transparency over automation
